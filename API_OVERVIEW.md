# API Онлайн-бібліотеки (демо)

Нижче наведено вигаданий приклад API, який міг би використовуватися у серверній версії проекту.

---

## **GET /api/books?genre={name}**

Отримати список книг певного жанру.

### Запит:

```
GET /api/books?genre=fantasy
```

### Відповідь:

```json
[
  { "id": 1, "title": "Гаррі Поттер", "author": "Дж. К. Ролінг" },
  { "id": 5, "title": "Володар перснів", "author": "Дж. Р. Р. Толкін" }
]
```

---

## **GET /api/book/{id}**

Отримати повну інформацію про книгу.

### Приклад запиту:

```
GET /api/book/1
```

### Приклад відповіді:

```json
{
  "id": 1,
  "title": "Гаррі Поттер",
  "author": "Дж. К. Ролінг",
  "description": "Книга про хлопчика-мага.",
  "genre": "Фентезі"
}
```

---

## **POST /api/favorites**

Додати книгу до обраного.

### Тіло запиту:

```json
{
  "bookId": 1
}
```

### Відповідь:

```json
{
  "message": "Book added to favorites"
}
```

---

## Коди стану

| Код | Значення          |
| --- | ----------------- |
| 200 | Успішно           |
| 201 | Створено          |
| 400 | Некоректні дані   |
| 404 | Книгу не знайдено |
| 500 | Серверна помилка  |
