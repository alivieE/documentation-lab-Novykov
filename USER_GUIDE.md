# Посібник користувача — Онлайн-бібліотека

## Як почати користуватися

1. Відкрийте додаток у браузері
2. На головній сторінці оберіть жанр (наприклад: «Фентезі», «Детектив», «Наукова література»)
3. Система покаже список книг, що відповідають жанру
4. Натисніть на книгу, щоб переглянути опис
5. Додайте книгу в «Обране», натиснувши кнопку ⭐


## Інтерфейс

### Головна панель

На головній панелі користувач бачить:

* Список жанрів
* Панель пошуку книг
* Кнопку “Обране”

### Показники книг

Для кожної книги відображаються:

* Назва книги
* Автор
* Короткий опис
* Кнопка ⭐ для додавання в обране

### Перегляд деталей

Клік на книгу відкриває детальну інформацію:

* Повний опис книги
* Жанр
* Дата публікації (якщо є)
* Кнопка повернення на головну

### Сортування та фільтри

Користувач може:

* Сортувати книги за жанрами
* Фільтрувати за популярністю або новинками

### Обране

У розділі **Обране** користувач бачить всі додані книги і може:

* Видалити книгу з обраного
* Переглянути деталі книги

### Сповіщення

Система може надсилати повідомлення, якщо:

* Обрана книга більше не доступна
* Додано нову книгу у вибраний жанр

## FAQ

### **1. Чому я не бачу книги після вибору жанру?**

Можливо, у цьому жанрі тимчасово немає даних або виникла помилка при завантаженні.

### **2. Дані зберігаються після перезавантаження?**

Так, список обраних зберігається у LocalStorage.

### **3. Чи потрібна реєстрація?**

Ні, система доступна без облікового запису.

### **4. Чи працює додаток на телефоні?**

Так, інтерфейс адаптований під мобільні пристрої.
