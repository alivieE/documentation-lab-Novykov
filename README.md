# Онлайн-бібліотека — система пошуку книг за жанрами

Онлайн-бібліотека — це веб-додаток, що дозволяє користувачам знаходити книги за жанрами, переглядати коротку інформацію та формувати власний список обраного.

## Що вирішує система?

Багато користувачів губляться в різноманітті книжок. Наш сервіс спрощує пошук: достатньо вибрати жанр, і система одразу покаже відповідні книги.

## Основні функції

* Пошук книг за жанрами
* Перегляд опису книги
* Додавання книг у «Обране»
* Адаптивний інтерфейс
* Збереження даних у браузері

## Для кого цей продукт?

* для студентів, яким потрібно швидко знайти літературу;
* для любителів книг, що хочуть мати зручну онлайн-полицю;
* для викладачів та бібліотекарів як інструмент швидкого пошуку.
