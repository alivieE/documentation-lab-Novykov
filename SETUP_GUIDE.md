# Інструкція з установки та запуску Онлайн-бібліотеки

## Необхідні інструменти

* Будь-який браузер
* Git або можливість завантажити ZIP
* (Опціонально) локальний сервер для розробки

## Встановлення

1. Клонування:

```
git clone https://github.com/your-username/online-library.git
```

2. Перехід у проєкт:

```
cd online-library
```

3. Відкрийте файл:

```
index.html
```

## Запуск

* Двічі натисніть по `index.html`
  або
* Запустіть локальний сервер:

```
npx serve
```

## Перевірка роботи

Після запуску має відобразитися список жанрів.
Перевірка працездатності:

* оберіть жанр → з’являється список книг;
* натисніть на книгу → відкривається опис;
* додайте книгу в обране → при перезавантаженні вона залишається.

Це означає, що система працює правильно.
